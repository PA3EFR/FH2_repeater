📋 1. Projectoverzicht

De FH-2 Repeater is een op Arduino NANO gebaseerde controller die automatisch CQ-berichten verstuurt via de Yaesu FH-2 Morsetoets op instelbare tijdsintervallen. Het systeem leest de FH-2 knoppenmatrix via analoge spanningsdetectie en stuurt via een 16-kanaals multiplexer (74HC4067) de juiste besturingspuls terug naar de FH-2.

Primair doel: Automatisch en herhaalbaar versturen van CQ-oproepen bij contest- en veldactivaties, zonder handmatig ingrijpen van de operator.

⚙️ 2. Functionele Beschrijving

Kernfunctionaliteit

Wanneer een operator op één van de knoppen 1–5 van de FH-2 drukt, detecteert de controller dit via de weerstandsladder op analoge pin A0. De controller stuurt vervolgens een puls via de multiplexer naar de FH-2. Indien de Repeat-schakelaar actief is, herhaalt het systeem deze puls automatisch op het door de potmeter ingestelde interval.

Gebruiksscenario

  1. Sluit de controller aan op de FH-2 via de 3,5mm stereo jack.
  2. Zet de Repeat-schakelaar op AAN (D7 = HIGH).
  3. Stel het gewenste interval in via de potmeter (10–60 seconden).
  4. Druk op knop 1–5 op de FH-2 (of via de passthrough).
  5. De controller stuurt het CQ-bericht automatisch op het ingestelde interval.
  6. Druk opnieuw op dezelfde knop om de cyclus te stoppen.

Stoppen van de intervalcyclus

  • Opnieuw indrukken van dezelfde knop (1–5)
  • Indrukken van een andere knop (directe override)
  • Omzetten van de Repeat-schakelaar naar UIT
Let op: Knoppen 6 t/m 12 (MEM, Links, Up, Rechts, Down, P/B, DEC) sturen wél een eenmalige puls maar starten geen intervalcyclus. Zij stoppen een eventueel actieve cyclus onmiddellijk.

74HC4067 Multiplexer

De 74HC4067 is een 16-kanaals bidirectionele analoge multiplexer/demultiplexer. Vier adreslijnen (S0–S3) selecteren één van de 16 kanalen. De ~EN-ingang (actief laag) schakelt de uitgang aan of uit. In dit project wordt de MUX gebruikt om de juiste besturingspuls naar de FH-2 te sturen zonder de knoppenmatrix te verstoren.

🔢 5. Knopdetectie — Spanningsreferentietabel

De FH-2 gebruikt een weerstandsladder waarbij elke knop een unieke spanning produceert op de analoge ingang A0. De controller meet deze spanning en vergelijkt met onderstaande referentietabel. Een tolerantievenster van ±0,15V zorgt voor betrouwbare detectie ondanks kleine voedings- of temperatuurschommelingen.

Detectie-algoritme

Het algoritme selecteert de knop waarvan de referentiespanning het dichtst bij de gemeten waarde ligt (best-match). Alleen als het minimale verschil kleiner is dan de tolerantie (0,15V) wordt de knop als ingedrukt beschouwd. 


  gemeten_spanning = (adc_waarde * 5.0) / 1023

  voor elke knop i:
      verschil = |referentie[i] - gemeten_spanning|
      if verschil < minimum_tot_nu:
          minimum_tot_nu = verschil
          beste_match = i

  if minimum_tot_nu < 0.15V:
      gedetecteerde_knop = beste_match
  else:
      gedetecteerde_knop = 0  (rust)

Debounce

Na een initiële detectie wordt 50ms gewacht en opnieuw gemeten. Alleen als beide metingen overeenkomen wordt de knop als geldig beschouwd. Dit elimineert valse detecties door contactstuitering.

Intervalinstelling via potmeter

De analoge waarde van A1 (0–1023) wordt lineair gemapt naar 10–60 seconden via Arduino's map() functie. De potmeter wordt elke lus-iteratie gelezen; bij een gewijzigde waarde wordt het nieuwe interval direct actief (ook tijdens een lopende cyclus). 


  interval = map(analogRead(A1), 0, 1023, 10, 60);
🔄 7. State Machine — Interval Cyclus

Het intervalrepeat-systeem werkt als een eenvoudige state machine met drie toestanden:

IDLE
(rust)
Knop 1–5 ingedrukt
+ Repeat AAN ➡
ACTIEF
(interval loopt)
Zelfde knop / andere knop
/ Repeat UIT ➡
GESTOPT
(→ IDLE)
ToestandConditie(s) om te verlatenActie
IDLE Knop 1–5 ingedrukt, Repeat = AAN → ACTIEF: eerste puls direct, timer gestart
ACTIEF Dezelfde knop opnieuw ingedrukt → GESTOPT / IDLE: Status LED uit
ACTIEF Andere knop ingedrukt Cyclus direct gestopt, nieuwe puls verstuurd
ACTIEF Repeat-schakelaar UIT gezet → GESTOPT / IDLE: geen verdere pulsen
ACTIEF Timer bereikt interval (10–60s) Automatische puls verstuurd, timer herstart
Opmerking: De Repeat-schakelaar (D7) blokkeert de automatische pulsafgifte maar stopt ook een actieve cyclus onmiddellijk. De eerste puls bij het starten van een cyclus wordt altijd direct verstuurd, ongeacht de ingestelde intervaltijd.

🔧 9. Bouw & Installatie

Bouwstappen

  1. Monteer de Arduino NANO op een prototype-printplaat of experimenteerplank.
  2. Verbind de 74HC4067 MUX-adreslijnen S0–S3 met NANO D5–D2 (let op: S0=D5, S1=D4, S2=D3, S3=D2).
  3. Verbind de ~EN pin van de MUX met NANO D6.
  4. Sluit de VCC en GND van de MUX aan op NANO 5V en GND.
  5. Verbind de 10kΩ potmeter: uiteinden naar 5V en GND, wiper naar NANO A1.
  6. Sluit de rode LED (met 220Ω in serie) aan van D9 naar GND.
  7. Sluit de groene LED (met 220Ω in serie) aan van D10 naar GND.
  8. Sluit de toggle schakelaar aan tussen D7 en GND.
  9. Verbind de FH-2 weerstandsladder-uitgang met NANO A0.
  10. Verbind de MUX-uitgang met de FH-2 besturingsingang via een 3,5mm stereo jack.

Firmware uploaden

  1. Download FH2_NANO.ino van de GitHub repository.
  2. Open het bestand in de Arduino IDE (versie 1.8.x of 2.x).
  3. Selecteer board: Arduino Nano (processor: ATmega328P).
  4. Selecteer de juiste COM-poort.
  5. Klik op Upload (→).
  6. Open de seriële monitor (9600 baud) om de opstart-header te controleren.
Let op met kloon-NANO's: Sommige Chinese NANO-klonen gebruiken een CH340G USB-chip. Installeer dan het juiste CH340-stuurprogramma. Selecteer in de Arduino IDE als processor: ATmega328P (Old Bootloader).

Eerste gebruik

  1. Verbind de controller met de FH-2.
  2. Zet de Repeat-schakelaar op AAN.
  3. Stel het interval in via de potmeter.
  4. Druk op knop 1–5 om een intervalcyclus te starten.
  5. Controleer de seriële monitor voor statusberichten.
  6. Druk opnieuw op dezelfde knop om de cyclus te stoppen.
Tip: Begin met een lang interval (potmeter max. = 60s) zodat u de werking rustig kunt observeren voordat u de CQ-cyclus in een contest gebruikt.
